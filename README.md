# Modern Todo List
Proyek Todo List yang dibuat sebagai bagian dari kurikulum [The Odin Project](https://www.theodinproject.com/).

## Fitur
- Membuat dan menghapus proyek.
- Menambah, mengedit, dan menghapus tugas (Todo).
- Fitur expand untuk melihat deskripsi tugas.
- Penyimpanan otomatis menggunakan Local Storage.
- UI yang modern dan responsif.

## Teknologi
- JavaScript (ES6 Modules)
- Webpack
- CSS3
- Date-fns (untuk manajemen tanggal)
